# NoteApp

A simple and efficient NoteApp built using pure **HTML**, **CSS**, and **JavaScript**. This application allows users to manage their notes with ease.  

## Features
- Add new notes.  
- Edit existing notes.  
- Delete notes.  
- Responsive design for a seamless experience on any device.
